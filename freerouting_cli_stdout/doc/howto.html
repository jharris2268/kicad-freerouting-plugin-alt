Freerouting_cli howto

From pcb-rnd

pcb-rnd from version 3.0.1 has freerouting_cli support from the external autorouter dialog or the extroute() action.

The dialog box is normally invoked from the menu: Connects/Automatic routing/External autorouter. TODO

From KiCad

  1. Start KiCad and open your project in Pcbnew.
  2. Export the PCB into Specctra DSN (File / Export... / Specctra DSN).
  3. Execute the router on the dsn file, get the result exported in a SES file.
  4. Go back to KiCad's Pcbnew and import the results (File / Import Specctra Session...).
