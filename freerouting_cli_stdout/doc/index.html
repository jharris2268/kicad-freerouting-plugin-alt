freerouting_cli

This is a fork of freerouting from github.com/freerouting. The intent of the project is to maintain a headless command line version with no GUI enabling it to be used as an external background router for pcb-rnd more easily.

Project source: svn://svn.repo.hu/freerouting_cli/trunk

Contact the authors: Email or chat

The java code of this fork is maintained by Erich S. Heinzle, the build system is maintained by Tibor 'Igor2' Palinkas.

This fork is being actively tested with the FOSS PCB layout editor pcb-rnd.
